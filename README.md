# AI LinkedIn Message Generator 🚀

An AI-powered web application that generates personalized LinkedIn connection messages based on role, company, and tone.

---

## 🔥 Features
- Generate personalized LinkedIn messages
- Choose tone: Friendly / Formal / Confident
- Clean and simple user interface
- Copy message instantly
- Helps improve networking outreach

---

## 🛠 Tech Stack
- Python (Flask)
- Gemini API
- HTML, CSS, JavaScript
